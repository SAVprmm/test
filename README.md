### How to use

* Windows `cmd.exe>`
  ```sh
  php app.php echo3 {verbose,overwrite}   [log_file=app.log]  {unlimited} [methods={create,update,delete}]   [paginate=50] {log}
  ```
  
* Linux `ssh#`
  ```sh
  php app.php echo3 \{verbose,overwrite\}   \[log_file=app.log\]  \{unlimited\} \[methods=\{create,update,delete\}\]   \[paginate=50\] \{log\}
  ```
**#!/bin/bash special symbols {}[]()$# and soo on are handled by POSIX**
